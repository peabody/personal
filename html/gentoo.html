Gentoo Linux Notes

I am currently experimenting with Gentoo Linux. It's kind of fun, but it's definitely a weird distro that takes some getting used to.

  • After an emerge it never hurts to do an env-update && source /etc/profile
  • Believe it or not there are binary versions of the of some popular software packages. They end in -bin
  • Always do an emerge -pv first to check and make sure that the package and the dependencies you're about to install have the use flags set that you want.
