Getting The Win-TV-HVR-950 Working in Hardy Heron

  • sudo apt-get install build-essential linux-headers dvb-utils gxine
  • wget http://linuxtv.org/hg/v4l-dvb/archive/tip.tar.gz
  • tar zxf tip.tar.gz
  • cd v4l-dvb*
  • make && sudo make install
  • wget http://www.steventoth.net/linux/xc5000/HVR-12x0-14x0-17x0_1_25_25271_WHQL.zip
  • unzip -j HVR-12x0-14x0-17x0_1_25_25271_WHQL.zip Driver85/hcw85bda.sys
  • linux/Documentation/video4linux/extract_xc3028.pl
  • sudo cp xc3028-v27.fw /lib/firmware
  • (Now plugin the stick)
  • mkdir -p ~/.xine
  • scan /usr/share/doc/dvb-utils/examples/scan/atsc/us-ATSC-center-frequencies-8VSB > ~/.xine/channels.conf
  • gxine dvb://'channel-name-here'
Read the ~/.xine/channels.conf file to see the names of the channels.
